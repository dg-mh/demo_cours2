git diff

Affiche les différences entre deux versions d'un ou plusieurs fichiers

git diff

Affiche les différences entre les fichiers actuels et ceux de l'index; en conséquence si tous vos fichiers modifiés sont dans la zone de staging git diff n'affichera pas de différence

git diff <commit1>...<commit2>

Affiche les différence dans les fichiers entre deux commits spécifiques. Le commit 1 doit être plus ancien que le commit 2

Ici, on peut indiquer les commits cible de différentes façons:

  • Nom de la branche
  • Nom du tag
  • Checksum d'un commit (SHA)
