git checkout

git checkout nous permet de déplacer HEAD d'un commit à l'autre.

git checkout [<commit>]

Ici, on peut indiquer le commit cible de différentes façons:

  • Nom de branche
  • Nom de tag
  • Checksum d'un commit (SHA)

git checkout -b <branche> [<commit>]

En plus de pointer HEAD sur le commit spécifié, crée une branche sur ce commit. Encore une fois on peut cibler le commit de différentes façons:

  • Nom de branche
  • Nom de tag
  • Checksum d'un commit (SHA)
